Website :

  • A collection of linked web pages that share a unique domain name.

  • A website is also known as a “web presence”.

  • Websites are made of a set of code or programming languages.

  • The most common languages used to build websites are: Hyper text markup language (HTML) helps browsers display webpage elements such as text, links and media. Cascading style sheets (CSS) define webpage styles such as colors, layouts and fonts

  • The first page of a website is called homepage.

  • Types:

    • Static Website : Content & Layout fix. No user interaction. Ex. HTML, CSS, JavaScript.
    • Dynamic Website : Content & Layout change. User interaction. Ex. HTML, CSS, JavaScript, Bootstrap, PHP.

    Webpage :

  • A single document on the web using a unique URL.

  • A web page can be accessed by entering a URL address into a browser's address bar.

  • A web page may contain text, graphics, and hyperlinks to other web pages and files.

  • Types:

    • Static web page
    • Dynamic web page
    • Blog page
    • Contact page
    • Home page
    • Media page

    Web Browser :

  • Access information on the World Wide Web

  • A web browser is an application for accessing websites.

  • The first web browser was called WorldWideWeb (WWW).

  • Types:

    • Microsoft Edge
    • Internet Explorer
    • Google Chrome
    • Mozilla Firefox
    • Apple Safari
    • Opera Mini

    Web Server :

  • A web server is a computer that runs websites.

  • A web server is a computer system with application software that delivers website content to the user.

  • The primary role of a web server is to store, process, and deliver requested information to users.

  • Types:

    • Apache
    • IIS
    • lighttpd
    • Jagsaw

    Internet :

  • INTERNET is a short form of Interconnected Network of all the Web Servers Worldwide. It is also called the World Wide Web or simply the Web.

  • Connects computers all over the world.

  • Through the Internet, people can share information and communicate from anywhere with an Internet connection.

  • Types:

    • Broadband Connection
    • Satellite Connection
    • Wireless Connection
    • Cellular
    • DSL (digital subscriber line)

    WWW :

  • It's also known as the web, World Wide Web or W3 or Mesh.

  • Tim Berners-Lee, a British scientist, invented the World Wide Web (WWW).

  • Access the content of websites / webpages from internet.

  • It much easier for people to get information, share, and communicate.

  • 3 protocols of WWW developed by Berners-Lee :

    • Hypertext Markup Language (HTML)
    • Uniform Resource Locator (URL)
    • Hypertext Transfer Protocol (HTTP)

    Protocol :

  • A set of rules for formatting and processing data.

  • Types :

    • HTTP (Hyper Text Transfer Protocol)
    • HTTPS (HyperText Transfer Protocol Secure)
    • FTP (File Transfer Protocol)
    • SMTP (Simple Mail Transfer Protocol)
    • PPP (Point-to-Point Protocol)
    • SFTP (Secure File Transfer Protocol)
    • TCP/IP (Transmission Control Protocol / Internet Protocol)

    HTTP & HTTPS :

  • Internet Protocols

    • HTTP

  • Hypertext Transfer Protocol

  • Transferring hypermedia documents, such as HTML.

  • The set of rules for transferring files -- such as text, images, sound, video and other multimedia files -- over the web.

  • Resources are exchanged between client devices and servers over the internet.

  • It is a protocol used to access the data on the World Wide Web (WWW).

    • HTTPS :

  • Hypertext Transfer Protocol Secure

  • An extension of the HTTP.
