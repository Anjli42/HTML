1. Are the HTML tags and elements the same thing ?

  • No, HTML tags and elements are not the same thing.

    • HTML Tags :

  • HTML tags are the fundamental building blocks of an HTML document.

  • Tags are enclosed in angle brackets (<>) and come in pairs - an opening tag and a closing tag.

  • The opening tag indicates the beginning of an element, and the closing tag marks the end of that element.

  • For example:- <p>This is a paragraph.</p>, <p> is the opening tag, and </p> is the closing tag.

    HTML Element :

  • An element includes both the tags and the content between them.

  • For example:- <p>This is a paragraph.</p>, <p> is the opening tag, "This is a paragraph." is the content, and </p> is the closing tag.

9. How is Cell Padding different from Cell Spacing? With Example.

    • Cell Padding :

  • Space between the cell edges (border) and the cell content.

  • Space between each cell.

  • Cellspacing can get subjected to more than one cell.

  • Cellspacing is an attribute.

  • Syntax:

    <table cellspacing="value" >.....</table>
    where, value determines the padding (space between adjacent cells)


10. How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

  • Both colspan and rowspan can be used together to create cells that span multiple rows and columns.

  • These attributes are useful for creating more complex layouts in tables.

    • Colspan :

  • Colspan specifies the number of columns a cell should span.

  • For example, if a cell has a colspan of 2, it will span across two columns in the table.

  • The colspan attribute is specified in the opening td tag of the cell to be merged, and the value of the attribute indicates the number of columns to span.

  • For example, if you want to merge three adjacent columns into a single cell, you would set the colspan attribute 3:

    <table BORDER="1">
    <tr>
    <td>Column 1</td>
    <td colspan="3">Merged cell spanning 3 columns</td>
    <td>Column 5</td>
    </tr>
    </table>

    • Rowspan :

  • Rowspan specifies the number of rows a cell should span.

  • For example, if a cell has a rowspan of 2, it will span across two rows in the table.

  • Rowspan is an HTML table attribute that allows a single table cell to span across multiple rows vertically. It is used to merge two or more adjacent rows into a single cell.

  • The rowspan attribute is specified in the opening td tag of the cell to be merged, and the value of the attribute indicates the number of rows to span.

  • For example, if you want to merge three adjacent rows into a single cell, you would set the rowspan attribute to 3:

    <table BORDER="1">
    <tr>
    <td rowspan="3">Merged cell spanning 3 rows</td>
    <td>Column 2</td>
    </tr>
    <tr>
    <td>Column 2</td>
    </tr>
    <tr>
    <td>Column 2</td>
    </tr>
    </table>

12. How to create a Hyperlink in HTML? With Example.

  • HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.

  • Syntax :

    <a href="url">link text</a>

  • The most important attribute of the element is the href attribute, which indicates the link's destination.

  • The link text is the part that will be visible to the reader.

  • Examples :

    <a href="https://www.w3schools.com/">Visit W3Schools.com!</a>

  • By default, links will appear as follows in all browsers:

    • An unvisited link is underlined and blue
    • A visited link is underlined and purple
    • An active link is underlined and red

    • HTML Links - The target Attribute

  • By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

  • The target attribute specifies where to open the linked document.

  • The target attribute can have one of the following values:

    • _self - Default. Opens the document in the same window/tab as it was clicked

      • Ex :

      <a href="https://www.w3schools.com/" target="_self">Visit W3Schools!</a>

    • _blank - Opens the document in a new window or tab

      • Ex :

      <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

    • _parent - Opens the document in the parent frame

      • Ex :

      <a href="https://www.w3schools.com/" target="_parent">Visit W3Schools!</a>

    • _top - Opens the document in the full body of the window

      • Ex :

      <a href="https://www.w3schools.com/" target="_top">Visit W3Schools!</a>

    • Absolute URLs vs. Relative URLs :

  • Both examples above are using an absolute URL (a full web address) in the href attribute. A local link (a link to a page within the same website) is specified with a relative URL (without the "https://www" part).

  • Ex :

    <h2>Absolute URLs</h2>
    <p><a href="https://www.w3.org/">W3C</a></p>
    <p><a href="https://www.google.com/">Google</a></p>

    <h2>Relative URLs</h2>
    <p><a href="html_images.asp">HTML Images</a></p>
    <p><a href="/css/default.asp">CSS Tutorial</a></p>


13. What is the use of an iframe tag? With Example.

  • The iframe in HTML stands for Inline Frame.

  • The ” iframe ” tag defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders.

  • An inline frame is used to embed another document within the current HTML document.

  • The HTML iframe name attribute is used to specify a reference for an <Iframe> element.

  • The iframe is basically used to show a webpage inside the current web page. The ‘ src ‘ attribute is used to specify the URL of the document that occupies the iframe.

  • src: It is used to specify the URL of the website to be loaded.

    title: It is good practice to include a title attribute so that screen readers can read the title to users.

  • Syntax:

    <iframe src="URL" title="description"> </iframe>

  • Attributes value: It contains a single value URL that specifies the URL of the document that is embedded in the iframe. There are two types of URL links which are listed below:

    • Absolute URL: It points to another webpage.
    • Relative URL: It points to other files of the same web page.

  • Accepted Attribute: The following attributes can be used with the <iframe> tag in HTML.

    • HTML <iframe> allow Attribute
    • HTML <iframe> allowfullscreen attribute
    • HTML <iframe> allowpaymentrequest attribute
    • HTML <iframe> height attribute
    • HTML <iframe> width attribute
    • HTML <iframe> loading attribute
    • HTML <iframe> scrolling attribute
    • HTML <iframe> name attribute
    • HTML <iframe> referrerpolicy attribute
    • HTML <iframe> sandbox attribute
    • HTML <iframe> src attribute
    • HTML <iframe> srcdoc attribute

  • Height and Width:

    • The height and width attributes are used to specify the size of the iframe. The attribute values are specified in pixels by default, but they can also be specified in percentages like ” 80% “.

    Example :

    <p>Content goes here</p>
    <iframe src= "https://google.com/.php" height="300" width="400"> </iframe>

  • Removing Border:

    • By default, iframe has a border around it. To remove the border, we must use the style attribute and use the CSS border property.

    Example :

    <p>Content goes here</p>
    <iframe src= "https://google.com/.php" height="300" width="400" style="border: none"> </iframe>

  • Border Style :

    • Changing the size, style, and color of the Iframe’s border:

    Example :

    <p>Content goes here</p>
    <iframe src= "https://google.com/.php" height="300" width="400" style="border: 4px solid orange"> </iframe>

  • Link :

    • An iframe can be used as the target frame for a link. The target attribute of the link must refer to the name attribute of the iframe.

    Example :

    <p>Content goes here</p>
    <iframe src= "https://google.com/.php" height="300" width="400" style="border: 4px solid orange"> </iframe>
    <p><a href="https://google.com/.php" target="iframe_a"> Link </a></p>

15. How to insert a picture into a background image of a web page? With Example.

  • In the body tag, specify a background image in the background attribute by passing the URL of the image or location path.

    Syntax:

    <body background = "URL or path" > Website Body </body>

  • Adding CSS styling properties.

    Syntax:

    <style>
    body {
        background-image:url(" URL of the image ");
    }
    </style>

  • The list of CSS property values associated with the background image:

    • url (Background Repeat) : URL to the background image. In case of more than one image, comma-separated list needs be provided.

      • <style>
      body {
      background-image: url("bg2.jpg"), url("bg3.png");
      background-repeat: no-repeat, repeat;
      background-color: #cccccc;
      }
      </style>

    • linear-gradient() : Sets a linear gradient as the background image. Needs at least 2 colors.

      • <style>
      #gradient {
      height: 200px;
      background-color: #cccccc;
      background-image: linear-gradient(red, yellow);
      }
      </style>

    • radial-gradient() : Sets a radial gradient as the background image. Needs at least 2 colors.

      • <style>
      #gradient1 {
      height: 300px;
      background-color: #cccccc;
      background-image: linear-gradient(red, blue, green);
      }
      </style>

    • repeating-linear-gradient() : Repeats a linear gradient

      • <style>
      #gradient1 {
      height: 300px;
      background-color: #cccccc;
      background-image: repeating-linear-gradient(red, blue 20%, green 30%);
      }
      </style>

    • repeating-radial-gradient() : Repeats a radial gradient

      • <style>
      #gradient1 {
      height: 200px;
      background-color: #cccccc;
      background-image: repeating-radial-gradient(red, blue 10%, green 20%);
      }
      </style>

    • initial : Sets the property to its default value
    • inherit : Inherits this property from its parent element


16. How are active links different from normal links?

  • The default color for normal and active links is blue in most of the websites.

  • Some browsers recognize an active link when the mouse cursor is placed over that link.

  • Whereas, others recognize active links when the link has the focus.

  • Those that don’t have a mouse cursor over that link is considered a normal link.

  • By default the links created are underlined.

  • a:link => This is a normal or unvisited link (BLUE).

  • a:visited => This is a link visited by user at least once (PURPLE).

  • a:active => This is a link that is just clicked (RED).

  • a:hover => This is a link when mouse hovers over it, it changes to a hand icon.

  • When a link is focused, it has an outline around it.

19. What is difference between HTML and XHTML?

    HTML

  • HTML stands for Hypertext Markup Language.

  • HTML is not a case-sensitive (lower or upper case) language.

  • It was developed by Tim Berners-Lee.

  • It was developed in 1991.

  • It is extended from SGML (Standard Generalized Markup Language).

  • The format is a document file format.

  • Doctype is not necessary to write at the top.

  • It is not necessary to close the tags in the order they are opened.

  • While using the attributes it is not necessary to mention quotes. For e.g. <Geeks>.

  • It can have a filename extension of.html or.htm.

    • XHTML

  • XHTML stands for Extensible Hypertext Markup Language.

  • HTML is a case-sensitive (lower case) language.

  • It was developed by W3C i.e World Wide Web Consortium.

  • It was released in 2000.

  • It is extended from XML and HTML.

  • The format is a markup language.

  • Doctype is very necessary to write at the top of the file.

  • It is necessary to close the tags in the order they are opened.

  • While using the attributes it is mandatory to mention quotes. For e.g. <Geeks=”GFG”>.

  • Its filename extension can be .xhtml,.xht, or .xml.
